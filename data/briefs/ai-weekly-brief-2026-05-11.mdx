## Major Launches

### AWS Agent Toolkit for Production AI Agents

AWS launched the **Agent Toolkit for AWS** — a production-ready suite of tools, MCP servers, and plugins designed specifically for AI coding agents. The toolkit includes 40+ skills across infrastructure-as-code, storage, analytics, containers, and AI services.

Key offering: the AWS MCP Server (now GA) provides fully-managed MCP access to any AWS service, with IAM guardrails, CloudWatch observability, and sandboxed execution. Three agent plugins bundle the MCP server with curated skills for quick installation.

**Why it matters:** AWS is treating AI coding agents as first-class consumers of its platform, not just human developers. This signals a shift toward agent-native cloud infrastructure.

### Adobe Productivity Agent

Adobe unveiled its **productivity agent** that orchestrates multiple models and tools to generate images, text, presentations, podcasts, and social content. It powers the new PDF Spaces feature in Acrobat Express and Acrobat Studio, turning static documents into interactive AI-assisted experiences.

The agent can create customized AI assistants that represent the sender's tone and intent — imagine a PDF that answers questions about itself in your voice.

### OpenAI GPT-Realtime-2 Voice API

OpenAI released three new voice models:

- **GPT-Realtime-2** — GPT-5-class reasoning for live voice interactions ($32/million audio input tokens)
- **GPT-Realtime-Translate** — Real-time translation across 70+ input languages at $0.034/minute
- **GPT-Realtime-Whisper** — Streaming speech-to-text at $0.017/minute

The pricing is aggressive — translation at one-third of a cent per minute undercuts most enterprise pipelines significantly. Early adopters include Zillow, Intercom, Priceline, and Deutsche Telekom.

## Enterprise Adoption

### Twilio Agent Connect (GA)

Twilio's **Agent Connect** is now generally available, providing a model-agnostic orchestration layer that connects AI runtimes (OpenAI, Anthropic, Bedrock, LangGraph) to Twilio's voice and messaging channels. Handles low-latency streaming, turn-taking, session management, and AI-to-human handoff.

Also launched: **Conversation Orchestrator** (GA) — bridges AI agents, human agents, and all communication channels with shared context.

### Workato Otto — Digital Teammate for Enterprise

Workato announced **Otto**, an autonomous AI agent embedded in Slack and Teams that executes work across enterprise systems. Powered by Workato Enterprise MCP, it requires no new integrations or security reviews. Over 1,000 users are already running Otto daily.

### Middleware OpsAI — AI SRE Agent

Middleware launched **OpsAI**, an AI-native Site Reliability Engineering agent that auto-resolves 80%+ of production issues. Built on full-stack observability (APM, logs, Kubernetes), it generates PR-ready code fixes and remediates incidents in real time.

## Trend: Agents Are the New SaaS

The pattern across all these launches is clear: **every platform is shipping its own AI agent layer**. AWS, Adobe, Twilio, Workato, ServiceNow, Middleware — all released agentic products in the same week.

The question is no longer "should you use AI agents?" but "which agent platform should you trust with your workflows?"
